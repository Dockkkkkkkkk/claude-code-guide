# 安装与初始配置 🟢

> **目标**：在自己电脑上成功安装并运行 Claude Code
> 

---

## 2.1 安装前准备

### 必须安装

**Node.js 18+**（Claude Code 运行依赖）

```bash
# 检查是否已安装
node --version   # 需要 v18 或更高
npm --version
```

如未安装，访问 [https://nodejs.org](https://nodejs.org) 下载 LTS 版本。

国内加速（npm 镜像）：

```bash
npm config set registry https://registry.npmmirror.com
```

**Git**（版本管理，强烈建议）

```bash
git --version   # 检查是否已安装
# 未安装：https://git-scm.com/downloads
```

---

## 2.2 四平台安装方式

### 🍎 macOS

```bash
# 方式一：npm（推荐，最通用）
npm install -g @anthropic-ai/claude-code

# 方式二：Homebrew
brew install --cask claude-code

# 方式三：Shell 一键脚本
curl -fsSL https://claude.ai/install.sh | sh
```

### 🐧 Linux

```bash
# 方式一：Shell 一键脚本（推荐）
curl -fsSL https://claude.ai/install.sh | sh

# 方式二：npm
npm install -g @anthropic-ai/claude-code
```

### 🪟 Windows

```powershell
# 方式一：WinGet（原生包管理器，推荐）
winget install Anthropic.ClaudeCode

# 方式二：PowerShell 脚本
irm https://claude.ai/install.ps1 | iex

# 方式三：npm（需先安装 Node.js）
npm install -g @anthropic-ai/claude-code
```

> ⚠️ Windows 用户注意：建议使用 **Windows Terminal** 或 **PowerShell 7+**，不推荐旧版 cmd.exe。
> 

### 🖥️ Desktop App（图形化界面）

直接到官网下载安装包：[https://claude.ai/download](https://claude.ai/download)

- macOS：`.dmg` 安装包
- Windows：`.exe` 安装程序
- Linux：`.AppImage` 或 `.deb`

> Desktop App 和 CLI 可以同时使用，共享配置文件。
> 

---

## 2.3 验证安装与首次登录

```bash
# 验证安装成功
claude --version
# 输出示例：Claude Code v2.1.84

# 首次启动（会打开浏览器进行授权）
claude

# 查看登录状态
claude auth status

# 退出登录
claude auth logout

# 重新登录
claude auth login
```

> 💡 国内用户如果无法访问 Anthropic 官方登录页，**先按第三章配置国产模型**，再回来使用 API Key 方式登录。
> 

---

## 2.4 更新与版本管理

```bash
# npm 安装的更新方式
npm update -g @anthropic-ai/claude-code

# Homebrew 更新（macOS）
brew upgrade claude-code

# WinGet 更新（Windows）
winget upgrade Anthropic.ClaudeCode

# 查看当前版本
claude --version

# 查看更新日志
claude changelog
```

> 建议保持更新——Claude Code 迭代非常快，每周都有新功能。
>

## 2.5 基础配置

### 配置文件位置

```
macOS / Linux:
  ~/.claude/settings.json     ← 全局配置
  ./claude/settings.json      ← 项目级配置

Windows:
  C:\Users\你的用户名\.claude\settings.json
```

### 默认模型配置

```json
{
  "defaultModel": "claude-sonnet-4-6-20261017",
  "env": {
    "ANTHROPIC_MODEL": "claude-sonnet-4-6-20261017"
  }
}
```

### 代理/网络配置（国内环境）

```bash
# 通过环境变量设置代理
export HTTPS_PROXY=http://127.0.0.1:7890
export HTTP_PROXY=http://127.0.0.1:7890

# 或在 settings.json 中配置
```

```json
{
  "env": {
    "HTTPS_PROXY": "http://127.0.0.1:7890",
    "HTTP_PROXY": "http://127.0.0.1:7890"
  }
}
```

> 💡 如果你使用国产模型（第三章内容），可以**不需要代理**，直接访问国内 API。
> 

### 常用 settings.json 选项速查

```json
{
  "theme": "dark",
  "autoUpdates": true,
  "telemetry": false,
  "defaultModel": "claude-sonnet-4-6-20261017",
  "env": {
    "API_TIMEOUT_MS": "600000",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": "1"
  }
}
```

## 🔬 实战案例：从零完成安装全流程（macOS示范）

```bash
# 第一步：安装 Node.js（如已安装跳过）
brew install node

# 第二步：安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 第三步：验证
claude --version

# 第四步：启动并登录（第一次会打开浏览器）
claude

# 第五步：执行第一个任务
# 在交互界面中输入：
你好，请告诉我当前目录下有哪些文件
```

如果看到 Claude 正确回答，安装成功！🎉
