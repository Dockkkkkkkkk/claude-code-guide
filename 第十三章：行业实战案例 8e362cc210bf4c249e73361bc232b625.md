## 13.1 电商平台——全链路商品上架自动化

**背景：**某中型电商公司，每次上架新品需要 6-8 个小时，包括：写文案、处理图片、内部审核等。

**核心提示词：**

```
我有一个新品 CSV 文件（product-data.csv），包含：
- product_name, sku, price, description（原始参数描述）

请依次完成：
1. 为每个商品写存量化略销文案（强调痛点+解决方案）
2. 为每个商品生成 5 个社交媒体备选标题
3. 检查内容是否符合公司审核标准（CLAUDE.md 里已定义）
4. 输出 formatted-products.json 和 publish-ready/ 目录
```

**效果：**上架时间从 6-8 小时降至 30 分钟。

## 13.2 内容创作——广告文案批量生产流水线

**背景：**广告公司每周需要为 20+ 客户分别出小红书/抖音/公众号等多平台物料。

**核心提示词：**

```
我是一家广告公司的文案策划师。
客户：某新能源公司，広州本地市场
机型：家用光伏安装服务

请为此客户生成本周内容日历：
- 小红书笔记 x3（生活方式切入，抱怪序列）
- 抖音口播脚本 x3（30s 左右）
- 公众号文章 x1（知识带货，深度了解多少钉子开开？）
- 决策者期广告文案 x2（强调投资回报）

须符合业主微信公众号风格（调研展示: mp.weixin.qq.com/xxx）
```

**效果：**一个策划师可小时内完成 7 个工作日的内容。

## 13.3 开发团队——旧项目快速现代化改造

**背景：**已有 5 年的 PHP 后台项目，需要迁移到 Node.js。

**核心提示词：**

```
项目目录：/legacy-php-app
目标语言：Node.js + Express + TypeScript
迁移方式：疑路并行，用新接口逐步替换

请帮我：
1. 分析 PHP 文件，提取所有 API 接口
2. 为每个接口创建 TypeScript 类型定义
3. 先迁移流量最大的用户登录模块
4. 确保新老接口返回格式完全一致
5. 为迁移后的接口写单元测试
```

**效果：**和一个资深工程师升天互公和的改造工作，续导出完成时间原日程的 70%。

## 13.4 数据分析师——自动化项目周报

**背景：**业务分析师每周期要生成项目鈀坠板，尾部周报，需要 3-4 小时。

**核心提示词：**

```
将以下三个数据源整合为周报：
- exports/weekly_sales.csv（销售数据）
- exports/user_retention.csv（用户留存）
- exports/ad_spend.csv（广告投放）

报告需包含：
1. 关键指标摘要（连环6周趋势）
2. 异常指标高亮标注
3. ROI 分析和预算建议
4. 下周行动建议（具体到一个责任人）

输出为 weekly-report-YYYY-MM-DD.md
```

**效果：**周报制作时间从 4 小时降至 20 分钟。

## 13.5 沃客服务——证据驱动的模型训练工作流

**背景：**沃客公司需要为不同客户训练定制化对话模型。

**核心提示词：**

```
咺沃数据: customer-service-logs.json（过去 6 个月的对话记录）

请：
1. 分析对话模式：找出 Top 20 常见问题类型
2. 为每种问题类型提取最佳回复模式
3. 出割异常处理（如加购抛弃、支付失败等效带情绪化场景）
4. 生成结构化的训练数据集 training-data.jsonl
5. 输出模型性能预测报告
```

**效果：**前期数据处理工作从 2 周缩短到 2 天。

## 13.6 法律行业——合同审查与风险识别

**背景：**小型律师事务所，需要审阅大量合同。

**核心提示词：**

```
合同文件：contracts/agreement-2026-031.pdf（已转换为文本）
客户身份：买方
项目类型：SaaS 订阅协议

请审查并标注：
1. 不利条款（限责、连带责任、任意修改子款）
2. 模糊文字（包括但不限于、合理念断等）
3. 缺少的关键条款（隐私、数据权限、退出机制）
4. 与行业标准合同的偏差

得出风险评分（1-10）和具体修改建议
```

**效果：**单张合同审查时间从 4 小时降至 30 分钟。

## 13.7 医疗健康——临床研究文献分析助手

**背景：**医学研究人员需要快速梳理大量文献。

**核心提示词：**

```
文献 PDF 目录：./papers/（已预处理为 txt）
研究问题：二型糖尿病患者运动干预的效果和安全性

请：
1. 扫描所有文献，筛选相关文献
2. 提取每篇的：样本量、干预类型、主要结果指标
3. 分析研究设计质量（RCT > 硬性 > 损性 > 病例报告）
4. 生成 PICO 表格和 刻朜广度
5. 输出系统评价报告和 元分析数据集
```

**效果：**文献返回工作从 2 周缩短到 3 天。

## 13.8 教育行业——个性化学习路径生成

**背景：**在线教育平台需要为不同基础的学生生成课程路径。

**核心提示词：**

```
学生评估结果：student-profile.json
  - 当前技能水平、学习目标、可用时间、学习风格

请为这个学生生成：
1. 8周个性化Python学习计划
2. 每天具体学什么，学多少时间
3. 推荐资源（符合学生的小红书风格）
4. 檢查节点和里程碑
5. 应对拖延的备选方案
```

**效果：**教师制作课程方案时间从 3 小时降至 10 分钟。

## 13.9 金融行业——投资有追踪和分析

**背景：**个人投资者需要监控进行中的投资组合。

**核心提示词：**

```
我目前的投资组合（CSV文件 portfolio.csv）

帮我：
1. 计算当前综合收益和比较基准收益
2. 分析行业和地區分散度
3. 对每个持仓搜集最新消息（使用 Brave Search MCP）
4. 识别可能需要低减的持仓
5. 出具此月投资建议报告
```

**效果：**投资组合分析时间从 4 小时降至 30 分钟。

## 13.10 人力资源——小型团队简历筛选工作流

**背景：**初创公司需要在 2 周内筛选 200+ 应聘者。

**核心提示词：**

```
岗位：全栈开发工程师
岗位 JD：job-description.md
应聘者文件夹：resumes/（PDF 已转换为 TXT）

请：
1. 对每个简历按下列标准打分（1-10）：
   - 技术匹配度、项目相关性、工作稳定性
2. 标注每个候选人的亮点和跨越
3. 按最终得分排序
4. 生成應聘者对比表 (CSV)
5. 为 Top 10 候选人生成面试问题清单
```

**效果：**简历筛选工作从 2 周天缩短至 3 小时。
