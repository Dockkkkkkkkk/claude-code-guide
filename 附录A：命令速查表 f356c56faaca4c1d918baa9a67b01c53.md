# 附录A：命令速查表

## 启动与基础

```bash
claude                          # 启动交互对话
claude "任务描述"                # 直接执行任务
claude --print "提示"           # 非交互输出
claude --resume                 # 恢复上次对话
claude --continue               # 继续上一次对话
claude --version                # 查看版本
```

## 模型与 API 设置

```bash
# 指定模型
claude --model claude-opus-4-5
claude --model deepseek-chat
claude --model kimi-k2.5

# 环境变量方式
export ANTHROPIC_API_KEY=sk-ant-xxxx
export ANTHROPIC_BASE_URL=https://api.deepseek.com/anthropic
export CLAUDE_MODEL=deepseek-chat
```

## 权限控制

```bash
# 指定允许的工具
claude --allowedTools "read_file,write_file,bash"

# 指定禁止的工具
claude --disallowedTools "delete_file"

# 跳过所有权限确认（开发环境）
claude --dangerously-skip-permissions

# 限制执行轮数
claude --max-turns 20
```

## 文件与目录

```bash
# 引用文件作为上下文
claude "分析 @src/main.ts"

# 添加目录到上下文
claude --add-dir ~/shared-skills

# 指定工作目录
claude --cwd /path/to/project
```

## 输出格式

```bash
claude --output-format text    # 纯文本（默认）
claude --output-format json    # JSON 格式
claude --output-format stream  # 流式输出
```

## 对话内斜杠命令

```
/help          # 帮助
/status        # 当前状态
/config        # 查看配置
/mcp           # MCP 服务器状态
/computer      # Computer Use 状态
/clear         # 清空当前对话
/exit          # 退出
```

## MCP 管理

```bash
claude mcp add <名称> -- <命令>          # 添加 stdio 类型
claude mcp add --transport http <名称> <URL>  # 添加 HTTP 类型
claude mcp add --scope user <名称> -- <命令>  # 全局范围
claude mcp list                          # 列出所有
claude mcp get <名称>                    # 查看详情
claude mcp remove <名称>                 # 删除
claude mcp add-from-claude-desktop       # 从 Desktop 导入
```

## 键盘快捷键（对话中）

| 快捷键 | 功能 |
| --- | --- |
| `ESC` | 取消当前操作 |
| `ESC ESC` | 退出 Agent 模式 |
| `Ctrl+C` | 强制退出 |
| `↑ / ↓` | 历史命令 |
| `Tab` | 自动补全 |
