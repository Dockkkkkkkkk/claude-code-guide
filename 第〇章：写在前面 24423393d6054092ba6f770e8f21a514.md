# 写在前面

## 这份教程的定位

这不是一份普通的「AI编程工具」教程。

Claude Code 的本质是一个**通用 AI Agent**——它能读文件、写代码、搜索网络、操控电脑、管理项目、自动执行定时任务……它住在你的终端里，是一个可以自主行动的超级助手。

本教程的目标：**从零开始，帮你真正掌握这个工具，并用它在实际工作中创造价值。**

> 当前版本：**Claude Code v2.1.84**（2026年3月26日更新）
>

## 关于「递归套娃五步法」

这是贯穿本教程的核心工作方法论，适用于所有复杂任务：

1. **第一步：明确目标** — 用一句话说清楚你想要什么结果
2. **第二步：拆解任务** — 让 Claude 先列出执行计划，你来审查
3. **第三步：逐步执行** — 分步确认，而不是一次性扔给 AI
4. **第四步：验证结果** — 每一步都检查输出是否符合预期
5. **第五步：迭代优化** — 发现问题及时调整，不要试图一次完美

> 💡 这五步法的精髓在于：**你是导演，AI 是执行者。** 你的判断力 × AI 的执行力 = 超强战斗力。
