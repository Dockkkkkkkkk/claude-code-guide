## 常见问题解答

**Q: Claude Code 彑4.5版本起不再免费？**

A: 正确。Claude Code 目前按 API 使用量计费。国内用户可用 DeepSeek 等国产模型大幅降低成本。

**Q: 所有人都需要学 Claude Code 吗？**

A: 不是。非开发者可重点学第1-5章（基础使用 + [CLAUDE.md](http://CLAUDE.md) + Skills），开发者建议全章学习。

**Q: 安全吗？代码会不会被泄露？**

A: Claude Code 的 Agent 模式不会上传代码。API 调用只发送提示词和相关片段。建议按第14章安全清单配置。

**Q: 国产模型能完全替代 Claude 吗？**

A: 对于日常开发任务，DeepSeek V3.2 + R1 组合可以达到 Claude Sonnet 80-90% 的质量，成本仄5-10x。

**Q: 如何最快学会 Claude Code？**

A: 直接用一个真实项目驱动学习，按第13章的行业案例找一个最接近你工作场景的例子尝试复现。
